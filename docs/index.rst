Installation
^^^^^^^^^^^^

Using Docker
------------

Requirements:

- git
- docker
- docker-compose

::

    git clone https://github.com/umrashrf/django-address-manager.git
    cd django-address-manager
    make start && make setup && make logs

::

    https://localhost:8000
    Username: admin
    Password: admin
